
# Rising Water Management System

## Project Description
The Rising Water Management System is a Flask-based web application that monitors rising water levels and displays flood alerts. It provides real-time information about water levels to help users stay informed.

## Features
- Displays current water level
- Shows flood alert status
- User-friendly interface
- Built using Flask, HTML, and CSS

# Rising Waters – Flood Prediction System

## Project Overview
Rising Waters is a machine learning-based flood prediction system that analyzes historical weather and rainfall data to predict flood risks. The system provides early warnings and helps authorities take preventive actions to reduce damage caused by floods.

## Objectives
- Predict flood occurrences accurately.
- Provide early warning alerts.
- Support disaster management teams.
- Improve public safety and preparedness.
